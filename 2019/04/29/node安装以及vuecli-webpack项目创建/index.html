node安装以及vuecli-webpack项目创建

  • 2019-04-29

安装node.js

> vue-cli3需要node8.11以上版本,这里推荐直接去官网下载最新的LTS版本

下载后自定义安装

1. 查看是否安装成功
    node -v
2. 查看npm
    npm -v
3. 设置包路径(可以默认)
    npm config set prefix "C:\Program Files\nodejs\npm_modules"
    npm config set cache "C:\Program Files\nodejs\npm_cache"
4. 安装cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org
5. 查看cnpm是否安装成功
    cnpm -v
    > 如果没有成功,可能是没有配置路径
    找到cnpm的安装路径
    一般是设置npm_modules的路径

6. 查看镜像是不是已经切换成淘宝源了
    nrm ls
    > 如果没有安装nrm的话 cnpm install -g nrm
7.安装webpack
    npm install webpack -g 或 cnpm install webpack -g
    安装带指定版本的webpack
    npm install webpack@3.6.0 -g或 cnpm install webpack@3.6.0 -g
    > 不建议安装4版本以上的,因为webpack-cli还要单独安装

安装vue-cli以及构建webpack项目

1.安装全局vue-cli插件
    npm install -g @vue/cli    
    cnpm install vue-cli -g

    cmd -> vue --version查看是否安装成功

提示版本低的时候(首先卸载再安装)
    npm uninstall -g vue-cli
    npm install -g @vue/cli
2.初始化项目
    vue init webpack projectname(或者vue init webpack-simple projectname)

    > 可能需要安装vue-init,按照提示安装即可

    例如:
        vue init webpack demo
        name demo
        description 随意
        author 随意
        vue build standalone
        vue-router yes
        eslint 可以选no
        set up unit tests no
        setup e2e tests.. no
        Should we run `npm install` for you after the project has been created? (recommended) (关键)
        选择 i will hanler... 即自己进入项目使用cnpm install 这样安装的比较快

    cd 到项目中
        cnpm install

2.在项目中安装element-ui
    cnpm i element-ui -S(首先安装element-ui进去)
3.使用
在main.js中引入
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

Vue.use(ElementUI);

4.创建.vue文件开始使用
